> [!IMPORTANT]
> ***File Name == CLASS NAME***

```terminal
javac Example.java-> compile program code 
java Example -> run compiled file 

```

```Example.java
class Example {
  public static void main(String args[]) { // *main function* -> Begin executing
                                           // 'public' can use outside of the class 
                                           // Java can only run class 'main'. class 'main'->use of running the others classes
                                           // 'args' receives any command-line arguments
    System.out.println("this is a simple java program ");
    // 'System' is a predefined class that provides access to the system 
    // 'out' -> output
  }
}
```
